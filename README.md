# AI Image Generator

A web-based AI image generator that creates high-quality images from text prompts using AI Horde. Users can select models, set image count and aspect ratio, and download generated images instantly. Features light/dark theme and random prompt suggestions for inspiration.

## Features
- Generate AI images from text prompts
- Choose AI model, image count, and aspect ratio
- Download generated images
- Light/Dark theme toggle
- Random prompt suggestions

## How to Use
1. Open `index.html` in your browser
2. Enter a text prompt or click "Random Prompt"
3. Select model, image count, and ratio
4. Click "Generate" to create images
5. Download images using the download icon

## Technologies
- HTML / CSS / JavaScript
- AI Horde API for image generation
